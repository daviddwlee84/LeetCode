# 226. Invert Binary Tree

## Description

Invert a binary tree.

**Example**:

Input:

```txt
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```

Output:

```txt
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
